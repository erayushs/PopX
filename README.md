# 🚀 PopX Authentication UI

A modern **React + TypeScript** authentication UI with **Sign In** and **Create Account** flows.  
Built with **TailwindCSS** for styling and **React Router** for navigation.

## ✨ Features

- 🔑 **Sign In Page**

  - Email & password input with validation
  - Disabled login button until form is valid
  - Inline error message for invalid email

- 📝 **Create Account Page**

  - Full name, phone, email, and password fields
  - Validation for required fields
  - Redirects to account settings on success

- 🎨 **Reusable Components**

  - `InputBox` → reusable input with label
  - `Button` → customizable with disabled state

- 🔄 **Routing**

  - Powered by `react-router-dom`
  - Smooth navigation between pages

- 📱 **Responsive Design**
  - Fully responsive with TailwindCSS
  - Works on desktop, tablet, and mobile
